# GET /budgets/{budgetSyncId}/export

**Resource:** [Settings](../resources/Settings.md)
**Exports the budget data as a zip file containing db.sqlite and metadata.json files.**
**Operation ID:** `get--budgets-{budgetSyncId}-export`

## Responses

| Status | Description |
|--------|-------------|
| 200 | The budget data as a zip file containing db.sqlite and metadata.json files |
| 404 | (reference) |
| 500 | (reference) |
