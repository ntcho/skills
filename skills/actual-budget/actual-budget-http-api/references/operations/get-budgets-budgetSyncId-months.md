# GET /budgets/{budgetSyncId}/months

**Resource:** [Budget Months](../resources/Budget-Months.md)
**Returns list of months for the budget associated with the sync id specified**
**Operation ID:** `get--budgets-{budgetSyncId}-months`

## Responses

| Status | Description |
|--------|-------------|
| 200 | The list of months for the specified budget |
| 404 | (reference) |
| 500 | (reference) |
