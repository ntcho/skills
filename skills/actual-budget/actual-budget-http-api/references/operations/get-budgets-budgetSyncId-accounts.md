# GET /budgets/{budgetSyncId}/accounts

**Resource:** [Accounts](../resources/Accounts.md)
**Returns list of accounts for the budget associated with the sync id specified**
**Operation ID:** `get--budgets-{budgetSyncId}-accounts`

## Responses

| Status | Description |
|--------|-------------|
| 200 | The list of accounts for the specified budget |
| 404 | (reference) |
| 500 | (reference) |
