# GET /budgets/{budgetSyncId}/schedules

**Resource:** [Schedules](../resources/Schedules.md)
**Returns list of schedules for the budget associated with the sync id specified**
**Operation ID:** `get--budgets-{budgetSyncId}-schedules`

## Responses

| Status | Description |
|--------|-------------|
| 200 | The list of schedules for the specified budget |
| 404 | (reference) |
| 500 | (reference) |
