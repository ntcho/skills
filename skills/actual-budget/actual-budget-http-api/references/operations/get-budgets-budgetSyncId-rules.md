# GET /budgets/{budgetSyncId}/rules

**Resource:** [Rules](../resources/Rules.md)
**Returns list of rules for the budget associated with the sync id specified**
**Operation ID:** `get--budgets-{budgetSyncId}-rules`

## Responses

| Status | Description |
|--------|-------------|
| 200 | The list of rules for the specified budget |
| 404 | (reference) |
| 500 | (reference) |
