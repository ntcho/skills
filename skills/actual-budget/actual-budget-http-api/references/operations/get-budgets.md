# GET /budgets

**Resource:** [Settings](../resources/Settings.md)
**Returns a list of all budget files either locally cached or on the remote server. Remote files have a state field and local files have an id field.**
**Operation ID:** `get--budgets`

## Responses

| Status | Description |
|--------|-------------|
| 200 | The list of budgets |
| 404 | (reference) |
| 500 | (reference) |
