# Categories

Endpoints for managing categories. See [Categories official documentation](https://actualbudget.org/docs/api/reference#categories)

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/budgets/{budgetSyncId}/categories` | Returns list of categories | [View](../operations/get-budgets-budgetSyncId-categories.md) |
| POST | `/budgets/{budgetSyncId}/categories` | Creates a category | [View](../operations/post-budgets-budgetSyncId-categories.md) |
| GET | `/budgets/{budgetSyncId}/categories/{categoryId}` | Returns a category information | [View](../operations/get-budgets-budgetSyncId-categories-categoryId.md) |
| DELETE | `/budgets/{budgetSyncId}/categories/{categoryId}` | Deletes a category | [View](../operations/delete-budgets-budgetSyncId-categories-categoryId.md) |
| PATCH | `/budgets/{budgetSyncId}/categories/{categoryId}` | Updates a category | [View](../operations/patch-budgets-budgetSyncId-categories-categoryId.md) |
| GET | `/budgets/{budgetSyncId}/categorygroups` | Returns list of category groups | [View](../operations/get-budgets-budgetSyncId-categorygroups.md) |
| POST | `/budgets/{budgetSyncId}/categorygroups` | Creates a category group | [View](../operations/post-budgets-budgetSyncId-categorygroups.md) |
| DELETE | `/budgets/{budgetSyncId}/categorygroups/{categoryGroupId}` | Deletes a category group | [View](../operations/delete-budgets-budgetSyncId-categorygroups-categoryGroupId.md) |
| PATCH | `/budgets/{budgetSyncId}/categorygroups/{categoryGroupId}` | Updates a category group | [View](../operations/patch-budgets-budgetSyncId-categorygroups-categoryGroupId.md) |
