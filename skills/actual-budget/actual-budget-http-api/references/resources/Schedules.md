# Schedules

Endpoints for managing schedules. See [Schedules official documentation](https://actualbudget.org/docs/api/reference#schedule)

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/budgets/{budgetSyncId}/schedules` | Returns list of schedules for the budget associated with the sync id specified | [View](../operations/get-budgets-budgetSyncId-schedules.md) |
| POST | `/budgets/{budgetSyncId}/schedules` | Creates a schedule | [View](../operations/post-budgets-budgetSyncId-schedules.md) |
| GET | `/budgets/{budgetSyncId}/schedules/{scheduleId}` | Returns a schedule | [View](../operations/get-budgets-budgetSyncId-schedules-scheduleId.md) |
| DELETE | `/budgets/{budgetSyncId}/schedules/{scheduleId}` | Deletes a schedule | [View](../operations/delete-budgets-budgetSyncId-schedules-scheduleId.md) |
| PATCH | `/budgets/{budgetSyncId}/schedules/{scheduleId}` | Updates a schedule | [View](../operations/patch-budgets-budgetSyncId-schedules-scheduleId.md) |
