# Payees

Endpoints for managing payees. See [Payees official documentation](https://actualbudget.org/docs/api/reference#payees)

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| GET | `/budgets/{budgetSyncId}/payees` | Returns list of payees | [View](../operations/get-budgets-budgetSyncId-payees.md) |
| POST | `/budgets/{budgetSyncId}/payees` | Creates a payee | [View](../operations/post-budgets-budgetSyncId-payees.md) |
| GET | `/budgets/{budgetSyncId}/payees/{payeeId}` | Returns a payee | [View](../operations/get-budgets-budgetSyncId-payees-payeeId.md) |
| DELETE | `/budgets/{budgetSyncId}/payees/{payeeId}` | Deletes a payee | [View](../operations/delete-budgets-budgetSyncId-payees-payeeId.md) |
| PATCH | `/budgets/{budgetSyncId}/payees/{payeeId}` | Updates a payee | [View](../operations/patch-budgets-budgetSyncId-payees-payeeId.md) |
| GET | `/budgets/{budgetSyncId}/payees/{payeeId}/rules` | Returns list of rules for a payee | [View](../operations/get-budgets-budgetSyncId-payees-payeeId-rules.md) |
| POST | `/budgets/{budgetSyncId}/payees/merge` | Merges payees | [View](../operations/post-budgets-budgetSyncId-payees-merge.md) |
