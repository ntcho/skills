# messagesclickdetail

Details about a message click event

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `ts` | integer | Yes | Unix timestamp when the click occurred |
| `url` | string | Yes | The URL that was clicked |
| `ip` | string | Yes | IP address of the clicker |
| `location` | string | Yes | Geographic location of the clicker |
| `ua` | string | Yes | User agent of the clicker |
