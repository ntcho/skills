# messagesopendetail

Details about a message open event

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `ts` | integer | Yes | Unix timestamp when the open occurred |
| `ip` | string | Yes | IP address of the opener |
| `location` | string | Yes | Geographic location of the opener |
| `ua` | string | Yes | User agent of the opener |
