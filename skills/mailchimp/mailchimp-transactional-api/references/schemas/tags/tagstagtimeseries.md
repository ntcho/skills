# tagstagtimeseries

Hourly statistics for a tag in a specific time period

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `sent` | integer | Yes | Number of emails sent during the hour |
| `hard_bounces` | integer | Yes | Number of emails that hard bounced during the hour |
| `soft_bounces` | integer | Yes | Number of emails that soft bounced during the hour |
| `rejects` | integer | Yes | Number of emails rejected during the hour |
| `complaints` | integer | Yes | Number of spam complaints received during the hour |
| `unsubs` | integer | Yes | Number of unsubscribe requests received during the hour |
| `opens` | integer | Yes | Number of emails opened during the hour |
| `unique_opens` | integer | Yes | Number of unique opens generated by messages sent during the hour |
| `clicks` | integer | Yes | Number of tracked URLs clicked during the hour |
| `unique_clicks` | integer | Yes | Number of unique clicks generated by messages sent during the hour |
| `time` | string | Yes | The hour as a UTC date string in YYYY-MM-DD HH:MM:SS format |
