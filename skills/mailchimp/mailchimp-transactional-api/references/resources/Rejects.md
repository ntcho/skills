# Rejects

Operations for managing email rejection blacklist

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/rejects/add` | Add email to rejection blacklist | [View](../operations/rejects-add.md) |
| POST | `/rejects/add-sms` | Add phone number to SMS rejection denylist | [View](../operations/rejects-add-sms.md) |
| POST | `/rejects/list` | List rejection blacklist entries | [View](../operations/rejects-list.md) |
| POST | `/rejects/list-sms` | List SMS rejection denylist entries | [View](../operations/rejects-list-sms.md) |
| POST | `/rejects/delete` | Delete email from rejection blacklist | [View](../operations/rejects-delete.md) |
| POST | `/rejects/delete-sms` | Delete phone number from SMS rejection denylist | [View](../operations/rejects-delete-sms.md) |
