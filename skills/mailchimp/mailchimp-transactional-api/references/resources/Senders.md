| POST | `/senders/verify-domain` | Verify a sender domain by email | [View](../operations/senders-verify-domain.md) |

| POST | `/senders/time-series` | Get hourly stats for a sender | [View](../operations/senders-time-series.md) |
| POST | `/senders/delete-domain` | Delete a sender domain | [View](../operations/senders-delete-domain.md) |
