# Tags

Operations related to managing and analyzing user-defined tags

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/tags/list` | List all user-defined tags | [View](../operations/tags-list.md) |
| POST | `/tags/delete` | Delete a tag permanently | [View](../operations/tags-delete.md) |
| POST | `/tags/info` | Get detailed information about a tag | [View](../operations/tags-info.md) |
| POST | `/tags/time-series` | Get time series data for a specific tag | [View](../operations/tags-time-series.md) |
| POST | `/tags/all-time-series` | Get time series data for all tags | [View](../operations/tags-all-time-series.md) |
