# Templates

Operations for managing email templates including creation, updating, publishing, and rendering

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/templates/add` | Add a new template | [View](../operations/templates-add.md) |
| POST | `/templates/info` | Get template information | [View](../operations/templates-info.md) |
| POST | `/templates/update` | Update an existing template | [View](../operations/templates-update.md) |
| POST | `/templates/publish` | Publish a template | [View](../operations/templates-publish.md) |
| POST | `/templates/delete` | Delete a template | [View](../operations/templates-delete.md) |
| POST | `/templates/list` | List all templates | [View](../operations/templates-list.md) |
| POST | `/templates/time-series` | Get template time series data | [View](../operations/templates-time-series.md) |
| POST | `/templates/render` | Render a template | [View](../operations/templates-render.md) |
