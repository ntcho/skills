# Users

Operations related to user account and credentials

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/users/info` | Return the information about the API-connected user | [View](../operations/users-info.md) |
| POST | `/users/ping` | Validate an API key and respond to a ping | [View](../operations/users-ping.md) |
| POST | `/users/ping2` | Validate an API key and respond to a ping (strict JSON parser version) | [View](../operations/users-ping2.md) |
| POST | `/users/senders` | Return the senders that have tried to use this account | [View](../operations/users-senders.md) |
