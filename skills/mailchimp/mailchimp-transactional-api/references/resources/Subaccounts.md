# Subaccounts

Operations for managing subaccounts including creation, configuration, and status control

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/subaccounts/list` | List subaccounts | [View](../operations/subaccounts-list.md) |
| POST | `/subaccounts/add` | Add a new subaccount | [View](../operations/subaccounts-add.md) |
| POST | `/subaccounts/info` | Get subaccount information | [View](../operations/subaccounts-info.md) |
| POST | `/subaccounts/update` | Update an existing subaccount | [View](../operations/subaccounts-update.md) |
| POST | `/subaccounts/delete` | Delete a subaccount | [View](../operations/subaccounts-delete.md) |
| POST | `/subaccounts/pause` | Pause a subaccount | [View](../operations/subaccounts-pause.md) |
| POST | `/subaccounts/resume` | Resume a paused subaccount | [View](../operations/subaccounts-resume.md) |
