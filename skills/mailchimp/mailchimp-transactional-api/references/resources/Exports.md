# Exports

Operations for exporting account data including rejection lists, allowlists, and activity history

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/exports/info` | Get export job information | [View](../operations/exports-info.md) |
| POST | `/exports/list` | List export jobs | [View](../operations/exports-list.md) |
| POST | `/exports/rejects` | Export rejection blacklist | [View](../operations/exports-rejects.md) |
| POST | `/exports/whitelist` | Export rejection allowlist (legacy) | [View](../operations/exports-whitelist.md) |
| POST | `/exports/allowlist` | Export rejection allowlist | [View](../operations/exports-allowlist.md) |
| POST | `/exports/activity` | Export activity history | [View](../operations/exports-activity.md) |
