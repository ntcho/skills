# IPs

Operations for managing dedicated IPs, IP pools, warmup processes, and custom DNS configuration

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|----------|
| POST | `/ips/list` | List dedicated IPs | [View](../operations/ips-list.md) |
| POST | `/ips/info` | Get dedicated IP information | [View](../operations/ips-info.md) |
| POST | `/ips/provision` | Request dedicated IP provisioning | [View](../operations/ips-provision.md) |
| POST | `/ips/start-warmup` | Start IP warmup process | [View](../operations/ips-start-warmup.md) |
| POST | `/ips/cancel-warmup` | Cancel IP warmup process | [View](../operations/ips-cancel-warmup.md) |
| POST | `/ips/set-pool` | Move IP to different pool | [View](../operations/ips-set-pool.md) |
| POST | `/ips/delete` | Delete dedicated IP | [View](../operations/ips-delete.md) |
| POST | `/ips/list-pools` | List dedicated IP pools | [View](../operations/ips-list-pools.md) |
| POST | `/ips/pool-info` | Get pool information | [View](../operations/ips-pool-info.md) |
| POST | `/ips/create-pool` | Create dedicated IP pool | [View](../operations/ips-create-pool.md) |
| POST | `/ips/delete-pool` | Delete dedicated IP pool | [View](../operations/ips-delete-pool.md) |
| POST | `/ips/check-custom-dns` | Test custom DNS configuration | [View](../operations/ips-check-custom-dns.md) |
| POST | `/ips/set-custom-dns` | Configure custom DNS for IP | [View](../operations/ips-set-custom-dns.md) |
