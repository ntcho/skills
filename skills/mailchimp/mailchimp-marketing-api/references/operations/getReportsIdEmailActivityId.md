# GET /reports/{campaign_id}/email-activity/{subscriber_hash}

**Resource:** [reports](../resources/reports.md)
**Get subscriber email activity**
**Operation ID:** `getReportsIdEmailActivityId`

Get a specific list member's activity in a campaign including opens, clicks, and bounces.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `campaign_id` | path | string | Yes | The unique id for the campaign. |
| `subscriber_hash` | path | string | Yes | The MD5 hash of the lowercase version of the list member's email address. |
| `since` | query | string | No | Restrict results to email activity events that occur after a specific time. Uses ISO 8601 time format: 2015-10-21T15:41:36+00:00. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
