# POST /lists/{list_id}/surveys/{survey_id}/actions/unpublish

**Resource:** [Surveys](../resources/Surveys.md)
**Unpublish a Survey**
**Operation ID:** `postListsIdSurveysIdActionsUnpublish`

Unpublish a survey that has been published.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `survey_id` | path | string | Yes | The ID of the survey. |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Survey Instance |
| default | An error generated by the Mailchimp API. |
