# GET /lists/{list_id}/members/{subscriber_hash}

**Resource:** [lists](../resources/lists.md)
**Get member info**
**Operation ID:** `getListsIdMembersId`

Get information about a specific list member, including a currently subscribed, unsubscribed, or bounced member.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `list_id` | path | string | Yes | The unique ID for the list. |
| `subscriber_hash` | path | string | Yes | The MD5 hash of the lowercase version of the list member's email address. This endpoint also accepts a list member's email address or contact_id. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
