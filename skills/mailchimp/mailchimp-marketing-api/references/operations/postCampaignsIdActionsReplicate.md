# POST /campaigns/{campaign_id}/actions/replicate

**Resource:** [campaigns](../resources/campaigns.md)
**Replicate campaign**
**Operation ID:** `postCampaignsIdActionsReplicate`

Replicate a campaign in saved or send status.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `campaign_id` | path | string | Yes | The unique id for the campaign. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
