# DELETE /ecommerce/stores/{store_id}/promo-rules/{promo_rule_id}/promo-codes/{promo_code_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Delete promo code**
**Operation ID:** `deleteEcommerceStoresIdPromocodesId`

Delete a promo code from a store.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `promo_rule_id` | path | string | Yes | The id for the promo rule of a store. |
| `promo_code_id` | path | string | Yes | The id for the promo code of a store. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
