# POST /campaigns/{campaign_id}/feedback

**Resource:** [campaigns](../resources/campaigns.md)
**Add campaign feedback**
**Operation ID:** `postCampaignsIdFeedback`

Add feedback on a specific campaign.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `campaign_id` | path | string | Yes | The unique id for the campaign. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
