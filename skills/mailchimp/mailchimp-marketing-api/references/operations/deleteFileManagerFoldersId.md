# DELETE /file-manager/folders/{folder_id}

**Resource:** [fileManager](../resources/fileManager.md)
**Delete folder**
**Operation ID:** `deleteFileManagerFoldersId`

Delete a specific folder in the File Manager.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `folder_id` | path | string | Yes | The unique id for the File Manager folder. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
