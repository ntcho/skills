# POST /ecommerce/stores/{store_id}/promo-rules/{promo_rule_id}/promo-codes

**Resource:** [ecommerce](../resources/ecommerce.md)
**Add promo code**
**Operation ID:** `postEcommerceStoresIdPromocodes`

Add a new promo code to a store.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `promo_rule_id` | path | string | Yes | The id for the promo rule of a store. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
