# GET /campaigns/{campaign_id}

**Resource:** [campaigns](../resources/campaigns.md)
**Get campaign info**
**Operation ID:** `getCampaignsId`

Get information about a specific campaign.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `campaign_id` | path | string | Yes | The unique id for the campaign. |
| `include_resend_shortcut_eligibility` | query | boolean | No | Return the `resend_shortcut_eligibility` field in the response, which tells you if the campaign is eligible for the various Campaign Resend Shortcuts offered. |
| `include_resend_shortcut_usage` | query | boolean | No | Return the `resend_shortcut_usage` field in the response.  This includes information about campaigns related by a shortcut. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
