# GET /audiences/{audience_id}/contacts

**Resource:** [audiences](../resources/audiences.md)
**Get Contacts**
**Operation ID:** `getAudienceContactList`

Get a list of omni-channel contacts for a given audience.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `count` | query | integer | No | The number of records to return. Default value is 10. Maximum value is 1000 |
| `cursor` | query | string | No | Paginate through a collection of records by setting the `cursor` parameter to a `next_cursor` attribute returned by a previous request. Default value fetches the first "page" of results. |
| `created_before` | query | string (date-time) | No | Restricts the response to contacts created at or before the specified time (inclusive). Uses ISO 8601 format: 2025-04-23T15:41:36+00:00. |
| `created_since` | query | string (date-time) | No | Restricts the response to contacts created after the specified time (exclusive). Uses ISO 8601 format: 2025-04-23T15:41:36+00:00. |
| `updated_before` | query | string (date-time) | No | Restricts the response to contacts updated at or before the specified time (inclusive). Uses ISO 8601 format: 2025-04-23T15:41:36+00:00. |
| `updated_since` | query | string (date-time) | No | Restricts the response to contacts updated after the specified time (exclusive). Uses ISO 8601 format: 2025-04-23T15:41:36+00:00. |
| `audience_id` | path | string | Yes | The unique ID for the audience. |
| `sort_field` | query | enum: created_at, updated_at | No | Specifies the field to sort the returned contacts by. |
| `sort_dir` | query | enum: ASC, DESC | No | Determines the order direction for sorted results. |
