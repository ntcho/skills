# POST /lists/{list_id}/segments

**Resource:** [lists](../resources/lists.md)
**Add segment**
**Operation ID:** `postListsIdSegments`

Create a new segment in a specific list.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
