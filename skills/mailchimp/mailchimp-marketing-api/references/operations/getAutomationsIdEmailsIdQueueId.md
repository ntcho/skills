# GET /automations/{workflow_id}/emails/{workflow_email_id}/queue/{subscriber_hash}

**Resource:** [automations](../resources/automations.md)
**Get automated email subscriber**
**Operation ID:** `getAutomationsIdEmailsIdQueueId`

Get information about a specific subscriber in a classic automation email queue.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `workflow_id` | path | string | Yes | The unique id for the Automation workflow. |
| `workflow_email_id` | path | string | Yes | The unique id for the Automation workflow email. |
| `subscriber_hash` | path | string | Yes | The MD5 hash of the lowercase version of the list member's email address. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
