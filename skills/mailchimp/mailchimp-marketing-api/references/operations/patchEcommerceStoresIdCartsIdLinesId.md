# PATCH /ecommerce/stores/{store_id}/carts/{cart_id}/lines/{line_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Update cart line item**
**Operation ID:** `patchEcommerceStoresIdCartsIdLinesId`

Update a specific cart line item.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `cart_id` | path | string | Yes | The id for the cart. |
| `line_id` | path | string | Yes | The id for the line item of a cart. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
