# DELETE /ecommerce/stores/{store_id}/products/{product_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Delete product**
**Operation ID:** `deleteEcommerceStoresIdProductsId`

Delete a product.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `product_id` | path | string | Yes | The id for the product of a store. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
