# POST /lists/{list_id}/members

**Resource:** [lists](../resources/lists.md)
**Add member to list**
**Operation ID:** `postListsIdMembers`

Add a new member to the list.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `skip_merge_validation` | query | boolean | No | If skip_merge_validation is true, member data will be accepted without merge field values, even if the merge field is usually required. This defaults to false. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
