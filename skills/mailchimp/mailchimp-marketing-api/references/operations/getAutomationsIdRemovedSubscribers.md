# GET /automations/{workflow_id}/removed-subscribers

**Resource:** [automations](../resources/automations.md)
**List subscribers removed from workflow**
**Operation ID:** `getAutomationsIdRemovedSubscribers`

Get information about subscribers who were removed from a classic automation workflow.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `workflow_id` | path | string | Yes | The unique id for the Automation workflow. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
