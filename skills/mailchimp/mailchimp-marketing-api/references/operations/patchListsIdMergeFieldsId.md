# PATCH /lists/{list_id}/merge-fields/{merge_id}

**Resource:** [lists](../resources/lists.md)
**Update merge field**
**Operation ID:** `patchListsIdMergeFieldsId`

Update a specific merge field.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `merge_id` | path | string | Yes | The id for the merge field. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`
