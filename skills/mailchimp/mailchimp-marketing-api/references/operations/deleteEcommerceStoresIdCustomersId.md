# DELETE /ecommerce/stores/{store_id}/customers/{customer_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Delete customer**
**Operation ID:** `deleteEcommerceStoresIdCustomersId`

Delete a customer from a store.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `customer_id` | path | string | Yes | The id for the customer of a store. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
