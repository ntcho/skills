# DELETE /connected-sites/{connected_site_id}

**Resource:** [connectedSites](../resources/connectedSites.md)
**Delete connected site**
**Operation ID:** `deleteConnectedSitesId`

Remove a connected site from your Mailchimp account.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `connected_site_id` | path | string | Yes | The unique identifier for the site. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
