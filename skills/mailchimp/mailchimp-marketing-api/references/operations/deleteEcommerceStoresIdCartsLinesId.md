# DELETE /ecommerce/stores/{store_id}/carts/{cart_id}/lines/{line_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Delete cart line item**
**Operation ID:** `deleteEcommerceStoresIdCartsLinesId`

Delete a specific cart line item.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `cart_id` | path | string | Yes | The id for the cart. |
| `line_id` | path | string | Yes | The id for the line item of a cart. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
