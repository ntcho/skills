# DELETE /templates/{template_id}

**Resource:** [templates](../resources/templates.md)
**Delete template**
**Operation ID:** `deleteTemplatesId`

Delete a specific template.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `template_id` | path | string | Yes | The unique id for the template. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
