# GET /templates

**Resource:** [templates](../resources/templates.md)
**List templates**
**Operation ID:** `getTemplates`

Get a list of an account's available templates.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `count` | query | integer | No | The number of records to return. Default value is 10. Maximum value is 1000 |
| `offset` | query | integer | No | Used for [pagination](https://mailchimp.com/developer/marketing/docs/methods-parameters/#pagination), this is the number of records from a collection to skip. Default value is 0. |
| `created_by` | query | string | No | The Mailchimp account user who created the template. |
| `since_date_created` | query | string | No | Restrict the response to templates created after the set date. Uses ISO 8601 time format: 2015-10-21T15:41:36+00:00. |
| `before_date_created` | query | string | No | Restrict the response to templates created before the set date. Uses ISO 8601 time format: 2015-10-21T15:41:36+00:00. |
| `type` | query | string | No | Limit results based on template type. |
| `category` | query | string | No | Limit results based on category. |
| `folder_id` | query | string | No | The unique folder id. |
| `sort_field` | query | enum: date_created, date_edited, name | No | Returns user templates sorted by the specified field. |
| `content_type` | query | enum: html, template, multichannel | No | Limit results based on how the template's content is put together. Only templates of type `user` can be filtered by `content_type`. If you want to retrieve saved templates created with the legacy email editor, then filter `content_type` to `template`. If you'd rather pull your saved templates for the new editor, filter to `multichannel`. For code your own templates, filter to `html`. |
| `sort_dir` | query | enum: ASC, DESC | No | Determines the order direction for sorted results. |

## Responses

|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
