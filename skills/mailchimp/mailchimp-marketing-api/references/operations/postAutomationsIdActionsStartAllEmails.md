# POST /automations/{workflow_id}/actions/start-all-emails

**Resource:** [automations](../resources/automations.md)
**Start automation emails**
**Operation ID:** `postAutomationsIdActionsStartAllEmails`

Start all emails in a classic automation workflow.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `workflow_id` | path | string | Yes | The unique id for the Automation workflow. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
