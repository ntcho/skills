# POST /audiences/{audience_id}/contacts/{contact_id}/actions/archive

**Resource:** [contacts](../resources/contacts.md)
**Archive Contact**
**Operation ID:** `postAudiencesContactsActionsArchive`

Archives a Contact.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `audience_id` | path | string | Yes | The unique ID for the audience. |
| `contact_id` | path | string | Yes | The unique id for the contact. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
