# DELETE /campaigns/{campaign_id}/feedback/{feedback_id}

**Resource:** [campaigns](../resources/campaigns.md)
**Delete campaign feedback message**
**Operation ID:** `deleteCampaignsIdFeedbackId`

Remove a specific feedback message for a campaign.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `campaign_id` | path | string | Yes | The unique id for the campaign. |
| `feedback_id` | path | string | Yes | The unique id for the feedback message. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
