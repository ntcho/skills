# POST /lists/{list_id}/surveys/{survey_id}/actions/create-email

**Resource:** [Surveys](../resources/Surveys.md)
**Create a Survey Campaign**
**Operation ID:** `postListsIdSurveysIdActionsCreateEmail`

Utilize the List ID and Survey ID to generate a Campaign that links to your survey.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `survey_id` | path | string | Yes | The ID of the survey. |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Campaign Instance |
| default | An error generated by the Mailchimp API. |
