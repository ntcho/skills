# DELETE /ecommerce/stores/{store_id}/products/{product_id}/variants/{variant_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Delete product variant**
**Operation ID:** `deleteEcommerceStoresIdProductsIdVariantsId`

Delete a product variant.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `product_id` | path | string | Yes | The id for the product of a store. |
| `variant_id` | path | string | Yes | The id for the product variant. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
