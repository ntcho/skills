# DELETE /ecommerce/stores/{store_id}/orders/{order_id}/lines/{line_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Delete order line item**
**Operation ID:** `deleteEcommerceStoresIdOrdersIdLinesId`

Delete a specific order line item.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `order_id` | path | string | Yes | The id for the order in a store. |
| `line_id` | path | string | Yes | The id for the line item of an order. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
