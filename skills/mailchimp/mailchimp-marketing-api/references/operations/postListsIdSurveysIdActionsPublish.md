# POST /lists/{list_id}/surveys/{survey_id}/actions/publish

**Resource:** [Surveys](../resources/Surveys.md)
**Publish a Survey**
**Operation ID:** `postListsIdSurveysIdActionsPublish`

Publish a survey that is in draft, unpublished, or has been previously published and edited.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `survey_id` | path | string | Yes | The ID of the survey. |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Survey Published |
| default | An error generated by the Mailchimp API. |
