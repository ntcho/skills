# DELETE /file-manager/files/{file_id}

**Resource:** [fileManager](../resources/fileManager.md)
**Delete file**
**Operation ID:** `deleteFileManagerFilesId`

Remove a specific file from the File Manager.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `file_id` | path | string | Yes | The unique id for the File Manager file. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
