# PATCH /ecommerce/stores/{store_id}/orders/{order_id}/lines/{line_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Update order line item**
**Operation ID:** `patchEcommerceStoresIdOrdersIdLinesId`

Update a specific order line item.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `order_id` | path | string | Yes | The id for the order in a store. |
| `line_id` | path | string | Yes | The id for the line item of an order. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`
