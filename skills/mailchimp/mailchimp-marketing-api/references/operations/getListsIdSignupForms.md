# GET /lists/{list_id}/signup-forms

**Resource:** [lists](../resources/lists.md)
**List signup forms**
**Operation ID:** `getListsIdSignupForms`

Get signup forms for a specific list.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
