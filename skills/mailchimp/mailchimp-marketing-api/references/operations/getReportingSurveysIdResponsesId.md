# GET /reporting/surveys/{survey_id}/responses/{response_id}

**Resource:** [reporting](../resources/reporting.md)
**Get survey response**
**Operation ID:** `getReportingSurveysIdResponsesId`

Get a single survey response.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `survey_id` | path | string | Yes | The ID of the survey. |
| `response_id` | path | string | Yes | The ID of the survey response. |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Get a single survey response. |
| default | An error generated by the Mailchimp API. |
