# GET /conversations/{conversation_id}/messages/{message_id}

**Resource:** [conversations](../resources/conversations.md)
**Get message**
**Operation ID:** `getConversationsIdMessagesId`
⚠️ **Deprecated**

Get an individual message in a conversation. Conversations has been deprecated in favor of Inbox and these endpoints don't include Inbox data. Past Conversations are still available via this endpoint, but new campaign replies and other Inbox messages aren’t available using this endpoint.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `conversation_id` | path | string | Yes | The unique id for the conversation. |
| `message_id` | path | string | Yes | The unique id for the conversation message. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
