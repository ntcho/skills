# GET /templates/{template_id}/default-content

**Resource:** [templates](../resources/templates.md)
**View default content**
**Operation ID:** `getTemplatesIdDefaultContent`

Get the sections that you can edit in a template, including each section's default content.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `template_id` | path | string | Yes | The unique id for the template. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
