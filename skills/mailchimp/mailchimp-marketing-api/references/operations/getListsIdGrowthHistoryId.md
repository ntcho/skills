# GET /lists/{list_id}/growth-history/{month}

**Resource:** [lists](../resources/lists.md)
**Get growth history by month**
**Operation ID:** `getListsIdGrowthHistoryId`

Get a summary of a specific list's growth activity for a specific month and year.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `list_id` | path | string | Yes | The unique ID for the list. |
| `month` | path | string | Yes | A specific month of list growth history. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
