# POST /ecommerce/stores/{store_id}/orders/{order_id}/lines

**Resource:** [ecommerce](../resources/ecommerce.md)
**Add order line item**
**Operation ID:** `postEcommerceStoresIdOrdersIdLines`

Add a new line item to an existing order.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `order_id` | path | string | Yes | The id for the order in a store. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
