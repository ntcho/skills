# PATCH /lists/{list_id}/webhooks/{webhook_id}

**Resource:** [lists](../resources/lists.md)
**Update webhook**
**Operation ID:** `patchListsIdWebhooksId`

Update the settings for an existing webhook.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `webhook_id` | path | string | Yes | The webhook's id. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
