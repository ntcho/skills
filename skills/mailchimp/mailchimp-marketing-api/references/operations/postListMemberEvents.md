# POST /lists/{list_id}/members/{subscriber_hash}/events

**Resource:** [lists](../resources/lists.md)
**Add event**
**Operation ID:** `postListMemberEvents`

Add an event for a list member.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `subscriber_hash` | path | string | Yes | The MD5 hash of the lowercase version of the list member's email address. This endpoint also accepts a list member's email address or contact_id. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
