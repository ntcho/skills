# DELETE /lists/{list_id}/interest-categories/{interest_category_id}/interests/{interest_id}

**Resource:** [lists](../resources/lists.md)
**Delete interest in category**
**Operation ID:** `deleteListsIdInterestCategoriesIdInterestsId`

Delete interests or group names in a specific category.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `interest_category_id` | path | string | Yes | The unique ID for the interest category. |
| `interest_id` | path | string | Yes | The specific interest or 'group name'. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
