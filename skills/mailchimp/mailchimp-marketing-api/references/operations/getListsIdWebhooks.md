# GET /lists/{list_id}/webhooks

**Resource:** [lists](../resources/lists.md)
**List webhooks**
**Operation ID:** `getListsIdWebhooks`

Get information about all webhooks for a specific list.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
