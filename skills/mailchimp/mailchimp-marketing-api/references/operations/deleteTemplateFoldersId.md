# DELETE /template-folders/{folder_id}

**Resource:** [templateFolders](../resources/templateFolders.md)
**Delete template folder**
**Operation ID:** `deleteTemplateFoldersId`

Delete a specific template folder, and mark all the templates in the folder as 'unfiled'.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `folder_id` | path | string | Yes | The unique id for the template folder. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
