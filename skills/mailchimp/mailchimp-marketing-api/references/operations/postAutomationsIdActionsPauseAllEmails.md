# POST /automations/{workflow_id}/actions/pause-all-emails

**Resource:** [automations](../resources/automations.md)
**Pause automation emails**
**Operation ID:** `postAutomationsIdActionsPauseAllEmails`

Pause all emails in a specific classic automation workflow.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `workflow_id` | path | string | Yes | The unique id for the Automation workflow. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
