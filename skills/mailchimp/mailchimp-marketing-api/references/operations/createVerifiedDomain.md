# POST /verified-domains

**Resource:** [verifiedDomains](../resources/verifiedDomains.md)
**Add domain to account**
**Operation ID:** `createVerifiedDomain`

Add a domain to the account.

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 | The newly-created domain. |
| default | An error generated by the Mailchimp API. |
