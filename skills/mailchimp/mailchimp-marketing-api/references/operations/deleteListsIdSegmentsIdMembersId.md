# DELETE /lists/{list_id}/segments/{segment_id}/members/{subscriber_hash}

**Resource:** [lists](../resources/lists.md)
**Remove list member from segment**
**Operation ID:** `deleteListsIdSegmentsIdMembersId`

Remove a member from the specified static segment.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `segment_id` | path | string | Yes | The unique id for the segment. |
| `subscriber_hash` | path | string | Yes | The MD5 hash of the lowercase version of the list member's email address. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
