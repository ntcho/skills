# DELETE /lists/{list_id}/webhooks/{webhook_id}

**Resource:** [lists](../resources/lists.md)
**Delete webhook**
**Operation ID:** `deleteListsIdWebhooksId`

Delete a specific webhook in a list.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `webhook_id` | path | string | Yes | The webhook's id. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
