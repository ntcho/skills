# PATCH /template-folders/{folder_id}

**Resource:** [templateFolders](../resources/templateFolders.md)
**Update template folder**
**Operation ID:** `patchTemplateFoldersId`

Update a specific folder used to organize templates.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `folder_id` | path | string | Yes | The unique id for the template folder. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
