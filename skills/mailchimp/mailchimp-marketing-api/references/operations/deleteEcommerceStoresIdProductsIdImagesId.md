# DELETE /ecommerce/stores/{store_id}/products/{product_id}/images/{image_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Delete product image**
**Operation ID:** `deleteEcommerceStoresIdProductsIdImagesId`

Delete a product image.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `product_id` | path | string | Yes | The id for the product of a store. |
| `image_id` | path | string | Yes | The id for the product image. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
