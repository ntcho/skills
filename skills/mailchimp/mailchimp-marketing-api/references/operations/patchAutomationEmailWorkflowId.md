# PATCH /automations/{workflow_id}/emails/{workflow_email_id}

**Resource:** [automations](../resources/automations.md)
**Update workflow email**
**Operation ID:** `patchAutomationEmailWorkflowId`

Update settings for a classic automation workflow email.  Only works with workflows of type: abandonedBrowse, abandonedCart, emailFollowup, or singleWelcome.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `workflow_id` | path | string | Yes | The unique id for the Automation workflow. |
| `workflow_email_id` | path | string | Yes | The unique id for the Automation workflow email. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
