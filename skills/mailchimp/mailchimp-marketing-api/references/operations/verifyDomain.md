# POST /verified-domains/{domain_name}/actions/verify

**Resource:** [verifiedDomains](../resources/verifiedDomains.md)
**Verify domain**
**Operation ID:** `verifyDomain`

Verify a domain for sending.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `domain_name` | path | string | Yes | The domain name. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 | The domain being verified for sending. |
| default | An error generated by the Mailchimp API. |
