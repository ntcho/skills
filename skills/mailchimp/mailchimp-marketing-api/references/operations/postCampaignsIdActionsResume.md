# POST /campaigns/{campaign_id}/actions/resume

**Resource:** [campaigns](../resources/campaigns.md)
**Resume rss campaign**
**Operation ID:** `postCampaignsIdActionsResume`

Resume an RSS-Driven campaign.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `campaign_id` | path | string | Yes | The unique id for the campaign. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
