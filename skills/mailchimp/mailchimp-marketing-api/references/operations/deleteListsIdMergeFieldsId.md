# DELETE /lists/{list_id}/merge-fields/{merge_id}

**Resource:** [lists](../resources/lists.md)
**Delete merge field**
**Operation ID:** `deleteListsIdMergeFieldsId`

Delete a specific merge field.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `merge_id` | path | string | Yes | The id for the merge field. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
