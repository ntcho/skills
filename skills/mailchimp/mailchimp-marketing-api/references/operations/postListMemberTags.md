# POST /lists/{list_id}/members/{subscriber_hash}/tags

**Resource:** [lists](../resources/lists.md)
**Add or remove member tags**
**Operation ID:** `postListMemberTags`

Add or remove tags from a list member. If a tag that does not exist is passed in and set as 'active', a new tag will be created.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `subscriber_hash` | path | string | Yes | The MD5 hash of the lowercase version of the list member's email address. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
