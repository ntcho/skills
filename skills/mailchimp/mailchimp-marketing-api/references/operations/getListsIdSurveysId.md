# GET /lists/{list_id}/surveys/{survey_id}

**Resource:** [lists](../resources/lists.md)
**Get survey**
**Operation ID:** `getListsIdSurveysId`

Get details about a specific survey.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `survey_id` | path | string | Yes | The ID of the survey. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
