# POST /campaigns/{campaign_id}/actions/unschedule

**Resource:** [campaigns](../resources/campaigns.md)
**Unschedule campaign**
**Operation ID:** `postCampaignsIdActionsUnschedule`

Unschedule a scheduled campaign that hasn't started sending.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `campaign_id` | path | string | Yes | The unique id for the campaign. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
