# DELETE /verified-domains/{domain_name}

**Resource:** [verifiedDomains](../resources/verifiedDomains.md)
**Delete domain**
**Operation ID:** `deleteVerifiedDomain`

Delete a verified domain from the account.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `domain_name` | path | string | Yes | The domain name. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
