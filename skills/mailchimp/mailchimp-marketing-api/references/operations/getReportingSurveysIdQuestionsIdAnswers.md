# GET /reporting/surveys/{survey_id}/questions/{question_id}/answers

**Resource:** [reporting](../resources/reporting.md)
**List answers for question**
**Operation ID:** `getReportingSurveysIdQuestionsIdAnswers`

Get answers for a survey question.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `survey_id` | path | string | Yes | The ID of the survey. |
| `question_id` | path | string | Yes | The ID of the survey question. |
| `respondent_familiarity_is` | query | enum: new, known, unknown | No | Filter survey responses by familiarity of the respondents. |

## Responses

| Status | Description |
|--------|-------------|
| 200 | List of Survey Question Answers |
| default | An error generated by the Mailchimp API. |
