# POST /automations/{workflow_id}/emails/{workflow_email_id}/queue

**Resource:** [automations](../resources/automations.md)
**Add subscriber to workflow email**
**Operation ID:** `postAutomationsIdEmailsIdQueue`

Manually add a subscriber to a workflow, bypassing the default trigger settings. You can also use this endpoint to trigger a series of automated emails in an API 3.0 workflow type.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `workflow_id` | path | string | Yes | The unique id for the Automation workflow. |
| `workflow_email_id` | path | string | Yes | The unique id for the Automation workflow email. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
