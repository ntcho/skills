# GET /reporting/facebook-ads/{outreach_id}/ecommerce-product-activity

**Resource:** [reporting](../resources/reporting.md)
**List facebook ecommerce report**
**Operation ID:** `getReportingFacebookAdsIdEcommerceProductActivity`

Get breakdown of product activity for an outreach.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `count` | query | integer | No | The number of records to return. Default value is 10. Maximum value is 1000 |
| `offset` | query | integer | No | Used for [pagination](https://mailchimp.com/developer/marketing/docs/methods-parameters/#pagination), this is the number of records from a collection to skip. Default value is 0. |
| `outreach_id` | path | string | Yes | The outreach id. |
| `sort_field` | query | enum: title, total_revenue, total_purchased | No | Returns files sorted by the specified field. |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Ecommerce Product Activity Instance |
| default | An error generated by the Mailchimp API. |
