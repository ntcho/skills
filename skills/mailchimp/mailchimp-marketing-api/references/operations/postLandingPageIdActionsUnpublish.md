# POST /landing-pages/{page_id}/actions/unpublish

**Resource:** [landingPages](../resources/landingPages.md)
**Unpublish landing page**
**Operation ID:** `postLandingPageIdActionsUnpublish`

Unpublish a landing page that is in draft or has been published.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `page_id` | path | string | Yes | The unique id for the page. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
