# GET /conversations/{conversation_id}/messages

**Resource:** [conversations](../resources/conversations.md)
**List messages**
**Operation ID:** `getConversationsIdMessages`
⚠️ **Deprecated**

Get messages from a specific conversation. Conversations has been deprecated in favor of Inbox and these endpoints don't include Inbox data. Past Conversations are still available via this endpoint, but new campaign replies and other Inbox messages aren’t available using this endpoint.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `fields` | query | string[] | No | A comma-separated list of fields to return. Reference parameters of sub-objects with dot notation. |
| `exclude_fields` | query | string[] | No | A comma-separated list of fields to exclude. Reference parameters of sub-objects with dot notation. |
| `conversation_id` | path | string | Yes | The unique id for the conversation. |
| `is_read` | query | enum: true, false | No | Whether a conversation message has been marked as read. |
| `before_timestamp` | query | string (date-time) | No | Restrict the response to messages created before the set time. Uses ISO 8601 time format: 2015-10-21T15:41:36+00:00. |
| `since_timestamp` | query | string (date-time) | No | Restrict the response to messages created after the set time. Uses ISO 8601 time format: 2015-10-21T15:41:36+00:00. |
