# POST /landing-pages/{page_id}/actions/publish

**Resource:** [landingPages](../resources/landingPages.md)
**Publish landing page**
**Operation ID:** `postLandingPageIdActionsPublish`

Publish a landing page that is in draft, unpublished, or has been previously published and edited.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `page_id` | path | string | Yes | The unique id for the page. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
