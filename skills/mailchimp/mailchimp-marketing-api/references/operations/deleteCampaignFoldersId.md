# DELETE /campaign-folders/{folder_id}

**Resource:** [campaignFolders](../resources/campaignFolders.md)
**Delete campaign folder**
**Operation ID:** `deleteCampaignFoldersId`

Delete a specific campaign folder, and mark all the campaigns in the folder as 'unfiled'.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `folder_id` | path | string | Yes | The unique id for the campaign folder. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
