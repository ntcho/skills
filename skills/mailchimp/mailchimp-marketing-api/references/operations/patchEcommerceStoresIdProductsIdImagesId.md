# PATCH /ecommerce/stores/{store_id}/products/{product_id}/images/{image_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Update product image**
**Operation ID:** `patchEcommerceStoresIdProductsIdImagesId`

Update a product image.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `product_id` | path | string | Yes | The id for the product of a store. |
| `image_id` | path | string | Yes | The id for the product image. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`
