# PATCH /campaign-folders/{folder_id}

**Resource:** [campaignFolders](../resources/campaignFolders.md)
**Update campaign folder**
**Operation ID:** `patchCampaignFoldersId`

Update a specific folder used to organize campaigns.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `folder_id` | path | string | Yes | The unique id for the campaign folder. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
