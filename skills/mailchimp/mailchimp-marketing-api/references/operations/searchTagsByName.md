# GET /lists/{list_id}/tag-search

**Resource:** [lists](../resources/lists.md)
**Search for tags on a list by name.**
**Operation ID:** `searchTagsByName`

Search for tags on a list by name. If no name is provided, will return all tags on the list.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `name` | query | string | No | The search query used to filter tags.  The search query will be compared to each tag as a prefix, so all tags that have a name starting with this field will be returned. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
