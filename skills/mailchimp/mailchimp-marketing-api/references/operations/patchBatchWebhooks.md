# PATCH /batch-webhooks/{batch_webhook_id}

**Resource:** [batchWebhooks](../resources/batchWebhooks.md)
**Update batch webhook**
**Operation ID:** `patchBatchWebhooks`

Update a webhook that will fire whenever any batch request completes processing.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `batch_webhook_id` | path | string | Yes | The unique id for the batch webhook. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
