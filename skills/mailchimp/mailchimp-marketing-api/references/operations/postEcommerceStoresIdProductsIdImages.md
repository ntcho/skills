# POST /ecommerce/stores/{store_id}/products/{product_id}/images

**Resource:** [ecommerce](../resources/ecommerce.md)
**Add product image**
**Operation ID:** `postEcommerceStoresIdProductsIdImages`

Add a new image to the product.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `product_id` | path | string | Yes | The id for the product of a store. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
