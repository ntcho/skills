# DELETE /ecommerce/stores/{store_id}/carts/{cart_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Delete cart**
**Operation ID:** `deleteEcommerceStoresIdCartsId`

Delete a cart.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `cart_id` | path | string | Yes | The id for the cart. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
