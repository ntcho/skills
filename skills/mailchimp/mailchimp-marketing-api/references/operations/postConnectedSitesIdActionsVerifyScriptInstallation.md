# POST /connected-sites/{connected_site_id}/actions/verify-script-installation

**Resource:** [connectedSites](../resources/connectedSites.md)
**Verify connected site script**
**Operation ID:** `postConnectedSitesIdActionsVerifyScriptInstallation`

Verify that the connected sites script has been installed, either via the script URL or fragment.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `connected_site_id` | path | string | Yes | The unique identifier for the site. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
