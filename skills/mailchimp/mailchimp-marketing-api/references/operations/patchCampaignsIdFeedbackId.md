# PATCH /campaigns/{campaign_id}/feedback/{feedback_id}

**Resource:** [campaigns](../resources/campaigns.md)
**Update campaign feedback message**
**Operation ID:** `patchCampaignsIdFeedbackId`

Update a specific feedback message for a campaign.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `campaign_id` | path | string | Yes | The unique id for the campaign. |
| `feedback_id` | path | string | Yes | The unique id for the feedback message. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
