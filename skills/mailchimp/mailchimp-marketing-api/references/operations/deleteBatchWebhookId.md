# DELETE /batch-webhooks/{batch_webhook_id}

**Resource:** [batchWebhooks](../resources/batchWebhooks.md)
**Delete batch webhook**
**Operation ID:** `deleteBatchWebhookId`

Remove a batch webhook. Webhooks will no longer be sent to the given URL.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `batch_webhook_id` | path | string | Yes | The unique id for the batch webhook. |

## Responses

| Status | Description |
|--------|-------------|
| 204 | Empty Response |
| default | An error generated by the Mailchimp API. |
