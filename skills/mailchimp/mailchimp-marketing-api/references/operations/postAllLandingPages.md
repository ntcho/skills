# POST /landing-pages

**Resource:** [landingPages](../resources/landingPages.md)
**Add landing page**
**Operation ID:** `postAllLandingPages`

Create an unpublished and contentless Mailchimp landing page.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `use_default_list` | query | boolean | No | Will create the Landing Page using the account's Default List instead of requiring a list_id. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
