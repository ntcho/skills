# POST /lists/{list_id}/segments/{segment_id}/members

**Resource:** [lists](../resources/lists.md)
**Add member to segment**
**Operation ID:** `postListsIdSegmentsIdMembers`

Add a member to a static segment.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `segment_id` | path | string | Yes | The unique id for the segment. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
