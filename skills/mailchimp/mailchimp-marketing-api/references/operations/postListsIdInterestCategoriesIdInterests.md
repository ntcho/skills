# POST /lists/{list_id}/interest-categories/{interest_category_id}/interests

**Resource:** [lists](../resources/lists.md)
**Add interest in category**
**Operation ID:** `postListsIdInterestCategoriesIdInterests`

Create a new interest or 'group name' for a specific category.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |
| `interest_category_id` | path | string | Yes | The unique ID for the interest category. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
