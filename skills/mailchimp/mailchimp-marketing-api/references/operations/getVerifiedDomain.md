# GET /verified-domains/{domain_name}

**Resource:** [verifiedDomains](../resources/verifiedDomains.md)
**Get domain info**
**Operation ID:** `getVerifiedDomain`

Get the details for a single domain on the account.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `domain_name` | path | string | Yes | The domain name. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
