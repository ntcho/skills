# GET /lists/{list_id}/surveys

**Resource:** [lists](../resources/lists.md)
**Get information about all surveys for a list**
**Operation ID:** `getListsIdSurveys`

Get information about all available surveys for a specific list.

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `list_id` | path | string | Yes | The unique ID for the list. |

## Responses

| Status | Description |
|--------|-------------|
| 200 |  |
| default | An error generated by the Mailchimp API. |
