# PATCH /ecommerce/stores/{store_id}/products/{product_id}

**Resource:** [ecommerce](../resources/ecommerce.md)
**Update product**
**Operation ID:** `patchEcommerceStoresIdProductsId`

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `store_id` | path | string | Yes | The store id. |
| `product_id` | path | string | Yes | The id for the product of a store. |

## Request Body

**Required:** Yes

**Content Types:** `application/json`
