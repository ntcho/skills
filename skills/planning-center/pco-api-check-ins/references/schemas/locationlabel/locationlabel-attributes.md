# locationlabel-attributes

Says how many of a given label to print for this location and
whether to print it for regulars, guests, and/or volunteers.


**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `created_at` | string (date-time) | No |  |
| `for_guest` | boolean | No |  |
| `for_regular` | boolean | No |  |
| `for_volunteer` | boolean | No |  |
| `quantity` | integer | No |  |
| `updated_at` | string (date-time) | No |  |
