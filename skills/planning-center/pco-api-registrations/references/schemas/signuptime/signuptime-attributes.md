# signuptime-attributes

`Signup_time`s are associated with a signup, which can have multiple signup times.


**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `all_day` | boolean | No | Whether or not the signup time is all day.
 |
| `created_at` | string (date-time) | No |  |
| `ends_at` | string (date-time) | No | End date and time of signup time.
 |
| `starts_at` | string (date-time) | No | Start date and time of signup time.
 |
| `updated_at` | string (date-time) | No |  |
