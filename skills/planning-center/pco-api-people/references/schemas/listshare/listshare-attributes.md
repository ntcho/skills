# listshare-attributes

A list share indicates who has access to edit a list.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `created_at` | string (date-time) | No |  |
| `group` | string | No | Possible values: `No Access`, `Viewer`, `Editor`, or `Manager` |
| `name` | string | No |  |
| `permission` | string | No | Possible values: `view` or `manage` |
