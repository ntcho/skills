# workflowshare-attributes

A workflow share defines who can access a workflow.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `group` | string | No | Possible values: `No Access`, `Viewer`, `Editor`, or `Manager` |
| `permission` | string | No | Possible values: `view`, `manage_cards`, or `manage` |
| `person_id` | string | No |  |
