# attendance-attributes

Individual event attendance for a person.


**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `attended` | boolean | No | Whether or not the person attended the event.
 |
| `role` | string | No | The role of the person at the time of event.


Possible values: `member`, `leader`, `visitor`, or `applicant` |
