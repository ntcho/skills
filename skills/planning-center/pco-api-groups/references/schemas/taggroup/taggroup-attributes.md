# taggroup-attributes

A way to group related tags.
For example you could have a "Life Stage" tag group
with tags like "Child", "Teen", "Adult", etc.


**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `display_publicly` | boolean | No | Whether or not this tag group is visible to the public on Church Center
 |
| `multiple_options_enabled` | boolean | No | Whether or not a group can belong to many tags within this tag group
 |
| `name` | string | No | The name of the tag group
 |
| `position` | integer | No | The position of the tag group in relation to other tag groups
 |
