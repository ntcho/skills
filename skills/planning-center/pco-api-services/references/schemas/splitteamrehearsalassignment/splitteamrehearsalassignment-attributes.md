# splitteamrehearsalassignment-attributes

For Rehearsal/Other Times, maps a Split Team to selected Time Preference Options. For example, used to assign 8am Ushers to 7:30am call time, and 11am Ushers to 10:30am call time.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `schedule_special_service_times` | boolean | No | Controls if the related rehearsal/other time is assigned when a person is scheduled to a split team service time that does not match a Time Preference Option |
