# resourcebooking-attributes

A specific booking of a room or resource for an event instance.


**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `created_at` | string (date-time) | No | UTC time at which the booking was created |
| `ends_at` | string (date-time) | No | UTC time at which usage of the booked room or resource ends |
| `quantity` | integer | No | The quantity of the rooms or resources booked |
| `starts_at` | string (date-time) | No | UTC time at which usage of the booked room or resource starts |
| `updated_at` | string (date-time) | No | UTC time at which the booking was updated |
