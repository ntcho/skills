# taggroup-attributes

A grouping of tags for organizational purposes.


**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `created_at` | string (date-time) | No | UTC time at which the tag group was created |
| `name` | string | No | The name of the tag group |
| `required` | boolean | No | - `true` indicates tag from this tag group must be applied when creating an event
 |
| `updated_at` | string (date-time) | No | UTC time at which the tag group was updated |
