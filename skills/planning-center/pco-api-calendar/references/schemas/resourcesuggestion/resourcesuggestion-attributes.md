# resourcesuggestion-attributes

A resource and quantity suggested by a room setup.


**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `created_at` | string (date-time) | No | UTC time at which the suggestion was created |
| `quantity` | integer | No | How many resources should be requested |
| `updated_at` | string (date-time) | No | UTC time at which the suggestion was updated |
