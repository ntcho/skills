# reporttemplate-attributes

A template for generating a report.


**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `body` | string | No | The contents of the report template |
| `created_at` | string | No | UTC time at which the report was created |
| `description` | string | No | A summarization of the report |
| `title` | string | No | The title of the report |
| `updated_at` | string | No | UTC time at which the report was updated |
